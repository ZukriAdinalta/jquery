# Mengenal Jquery
-> pustaka javascript yang digunakan untuk mempercepat, memperingkas serta sederhana manipulasi document HTML, penanganan event, animasi dan interaksi AJAX untuk mempercepat pengembangan web.

1. Syntak JQUERY
$(selector).action()
JQuery(selector).action();
- $ => simbol dolar mendefinisikan fungsi Jquery
- Selector => menunjukan elemen yg dipilih bisa berupa tag, nama class atau id elemen.
- action => aksi yg diberikan terhadap elemen yg dipilih

- https://jquery.com/download/

- copy coding simpan di jquery.js
